Recent Work

Harvard Square Homeless Shelter App

I am leading a team of 12 students in developing a mobile app for the HSHS using React Native and Firebase. It enables users to view and log interactions with guests, maintain a shared to-do list, and much more.

Red or Blue

I am making a 2D puzzle platformer in Unity where platforms, enemies and traps are either colored red or blue, and only one color is visible and can be interacted with at once. (codebase)

Microsoft Avatar (Finished top 10 in the Millennials category at 2017 Microsoft Global Hackathon)

At Microsoft Research, I led a team of five in creating a style-transfer web app based on Deep Image Analogy technique. It enables users to create stylized digital avatars using any selfie. I planned the project from scratch and designed and implemented the front-end using React. (No public demo available)

In depth analysis of the Great Firewall of China

As a research project, I studied the censorship techniques of the GFW and potential countermeasures, using captured packets for experiment.

MyDea

I co-created an Android app that lets users share ideas for projects they are interested in, which other users can view, comment, vote on, and join. Used Node.js and PostgreSQL for backend. (codebase)

Drizzle (Winner of Bose Prize at PolyHack 2016)

I made a web app using Python, Django, D3 JS Library, and the Bose SoundTouch API that enables users to mix and create customized ambient noise tracks with an innovative clock-like interface.
